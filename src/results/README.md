# Results

This directory stores results generated by experiments implemented in `src/`.

## Recommended Structure

```text
results/
  tabular_letterenv/
    simple/
    one_hot/
    numerical/
    figures/
  dqn_letterenv/
    simple/
    one_hot/
    numerical/
```

## Run Metadata

Each run directory should record at minimum:

- code or configuration version
- random seed
- encoding type
- total training timesteps
- evaluation success metrics
- any explicit deviation from the baseline protocol
